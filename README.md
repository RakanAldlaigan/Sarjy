# Sarjy

A voice assistant web app for capturing notes, tasks, and reminders by voice,
with cross-session memory and Google Calendar integration.

URL: https://sarjy-mauve.vercel.app


## Tech stack

- **Frontend**: Next.js
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL via Supabase
- **Auth**: Google OAuth
- **STT**: Deepgram
- **TTS**: ElevenLabs
- **LLM**: Google Gemini 2.5 Flash
